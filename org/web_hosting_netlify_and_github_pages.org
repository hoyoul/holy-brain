* web 개발
webpage를 만드는 방식은 2가지다.  html과 js로 page를 만든다. 필요한
data는 rest api로 가져와서 보여준다. 

만들어서 모든 사람에게 보여줄려면 webserver에 올려야
한다. 이것을 deploy라고 부른다. deploy할 web server를 예전에는
만들었다.
